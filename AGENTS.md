## Validação

Este é um projeto estático feito com HTML, CSS e JavaScript puro.

Não execute comandos npm, pois o projeto não utiliza Node.js e não possui package.json.

Após cada etapa:

1. Verifique se o HTML, CSS e JavaScript continuam funcionando.
2. Teste o site pelo Live Server.
3. Verifique o console do navegador.
4. Teste em diferentes larguras de tela.
5. Verifique links, botões, menu e troca de idiomas.
6. Informe todos os arquivos alterados.
7. Resuma as mudanças realizadas.
8. Não adicione Node.js, frameworks ou ferramentas de build sem autorização explícita.
